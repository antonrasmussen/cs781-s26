## Assignment layout

Each assignment lives in its own subfolder and typically includes:

- **README.md** – Setup, how to run, and contents
- **Data** – Input dataset(s)
- **Notebook(s)** – Code and results
- **report.md** (or similar) – Written report (methods, results, limitations)
- **requirements.txt** – Python dependencies for that assignment

Run instructions and data paths are relative to the assignment folder (e.g. run the notebook from `diabetes/` so that `diabetes.csv` is in the same directory).
